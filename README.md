# PrajnaAI

**PrajnaAI** is a beginner-level educational web platform developed as a **BCA Black Book project**. The purpose of this website is to help students easily **import, export, and manage study materials** such as notes, documents, and other academic resources.

The platform aims to make learning resources more **organized, accessible, and easy to share** among students. PrajnaAI focuses on creating a simple and user-friendly system where study materials can be uploaded, stored, and downloaded efficiently.

This project demonstrates basic concepts of **web development and educational resource management**, making it useful for students who want a centralized place to access and share study materials.
